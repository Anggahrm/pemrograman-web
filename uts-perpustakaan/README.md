# BacaKita - Sistem Informasi Perpustakaan Digital

BacaKita adalah website front-end bertema sistem informasi perpustakaan digital. Project ini dibuat untuk memenuhi tugas UTS Praktikum Pemrograman Web.

## Konsep UI/UX

Tampilan website mengikuti gaya Academic Modernity: layout bersih, warna deep indigo, surface putih/off-white, card buku rasio 2:3, navigasi rapi, dan tipografi Inter untuk kesan akademik modern.

## Halaman Website

- `index.html` - halaman utama berisi hero, statistik, layanan, dan koleksi unggulan.
- `katalog.html` - halaman katalog berisi 18 buku nyata dengan search, filter kategori, sorting, status buku, cover buku, dan pagination aktif.
- `kontak.html` - halaman kontak berisi profil layanan, informasi kontak, jam layanan, dan form kritik/saran.

## Teknologi

- HTML
- CSS external
- JavaScript

## Fitur

- Navigasi antar halaman menggunakan relative link.
- Struktur HTML menggunakan `header`, `nav`, `main`, dan `footer`.
- Styling memakai external CSS di `css/style.css`.
- Desain responsive untuk desktop, tablet, dan mobile.
- Katalog buku berbentuk card dengan cover visual.
- Search buku berdasarkan judul, penulis, atau kategori.
- Filter kategori dan sorting katalog menggunakan JavaScript.
- Pagination katalog aktif dengan 6 buku per halaman.
- Validasi sederhana pada form kontak.

## Cara Menjalankan

Buka file `index.html` menggunakan browser, lalu gunakan menu navigasi untuk berpindah halaman.

## GitHub Pages

1. Upload semua file ke repository GitHub.
2. Buka menu Settings repository.
3. Pilih Pages.
4. Pada bagian Source, pilih branch utama dan folder root.
5. Simpan pengaturan dan buka link GitHub Pages yang tersedia.
